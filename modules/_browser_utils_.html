newPage

  • newPage(config: any, chrome_args?: string[]): Page
    • Defined in browser_utils.js:36

    Launch a new browser with puppeteer, with a new page (=Tab). The browser is completely isolated from any other calls. Most interactions will be with the page, but you can get the browser using await page.browser();. For more information about the page object, see the puppeteer API documentation.

    example

await page.goto('https://example.org/');
await waitForText(page, 'More information');
await closePage(page);

    Parameters

      The pentf configuration object.

    • Default value chrome_args: string[] = []

    Returns Page

    The puppeteer page handle.

waitForVisible

  • waitForVisible(page: Page, selector: string, __namedParameters?: {}): Promise<ElementHandle<Element>>
    • Defined in browser_utils.js:192

    Wait for an element matched by a CSS query selector to become visible. Visible means the element has neither display:none nor visibility:hidden. Elements outside the current viewport (e.g. you'd need to scroll) and hidden with CSS trickery (opacity, overlaid with z-index, or permanently positioned outside the viewport) count as visible.

    Parameters

    • page: Page

      puppeteer page object.

    • selector: string

      Query selector, e.g. div > a[href="/"]:visited

    • Default value __namedParameters: {} = {}

    Returns Promise<ElementHandle<Element>>

    A handle to the found element.
